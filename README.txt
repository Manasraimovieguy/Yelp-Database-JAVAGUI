The following project for Assignment 3 creates a GUI that connects to Oracle Database 
via JDBC to create an application using Swing.

To compile the project on Netbeans, the instructions are provided below

1. Intall and connect to Oracle Database
2. Create the necessary tables and indexes on Oracle
3. Run the populate.java file to populate the database using the given JSON files
4. After populating the database, run HW3.java to execute the GUI for Business Search and start executing queries.
5. To execute User Search, run Users.java to execute the GUI for User search
